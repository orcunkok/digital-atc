Altitude wording guide (for readbacks)

- 500 -> "five hundred"
- 1,000 -> "one thousand"
- 1,500 -> "one thousand five hundred"
- 2,000 -> "two thousand"
- 10,000 -> "one zero thousand" is not standard; prefer "ten thousand"
- 12,500 -> "twelve thousand five hundred"

Speed:
- Pronounce as numbers: e.g., 110 kt -> "one one zero knots" (or keep unspoken in readback if not assigned)

Headings (deg, 3-digits if needed):
- 030 -> "zero three zero"
- 280 -> "two eight zero"
